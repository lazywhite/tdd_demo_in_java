## Introduction
通过编写一个元音字符处理函数来践行TDD思路    
[参考文档](http://icodeit.org/2014/11/tdd-step-by-step/)    

```
流程
    1. 编写新测试用例, 运行失败(没有任何实现)
    2. 以最简单的方式实现, 并通过测试
    3. code review, 重构代码和testCase, 并通过测试
    4. 开始新一轮迭代
```

```
函数要求   
 
    输入一个非元音字符，并预期返回字符本身
    输入一个元音，并预期返回mommy
    输入一个元音超过30%的字符串，并将元音替换为mommy
    输入一个元音超过30%，并且存在连续元音的字符串，连续的元音只被替换一次

```
